OOP Design

1. Understand the Problem

• Set up game with board and players
• Game loop
• Draw the board
• Choose random word
• Get guess from player
• Validate letter
• Check for correct or wrong letter
• Finish game if player has guessed the word or completed the body

2. Pseudocode the High Level Approach

Set up game
   Create game board
Create players
   Start game loop
   Render board
   Generate random word
   Display length of word to players
   Get letters from guessers and validate 
   If letter is correct, display on letter line
   If wrong, draw body part on hanger
   Display wrong guess on the side
If game should end
   Display victory or loss message
   Stop looping
Else
   Switch to next player and keep looping

3. Assign Actions to Objects

Set up game [hangman]
   Create game board [board]
Create players [player]
   Start game loop [hangman]
   Render board [board]
   Generate random word [hangman]
   Display length of word to players [board]
   Get letters from guessers and validate [player]
   If letter is correct, display on letter line [board]
   If wrong, draw body part on hanger [board]
   Display wrong guess on the side
If game should end [hangman]
   Display victory or loss message
   Stop looping
Else
   Switch to next player and keep looping [hangman]

4. Make Javascript-ish Pseudocode

class Hangman
    initialize
        set up board
        set up players
        set up starting player

    play
        loop infinitely
            call the board rendering method
            generate random word
            display length of word to players
            get letters from guessers and validate

    check_game_over
        check_victory
        check_loss

    check_victory
        if board says word is complete
            display victory message

    check_loss
        if board says body is complete
            display loss message
