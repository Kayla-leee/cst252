LAB 16: LIBRARIES & INTRO TO JQUERY

LAB SUMMARY

This lab was focused on experimenting with jQuery, as we have just started learning about it. We were to create and style a button using jQuery only, and not with HTML, Javascript, and CSS like we have been doing for the past few labs. After creating the button, we added an alert function to the button so it would prompt a pop-up window in the browser.

CHALLENGES

There were no challenges with this lab, expect for a short, minor struggle in trying to go about creating the button without using HTML. With the lecture notes and help from each other, we were able to breeze through this lab.
