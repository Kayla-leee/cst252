Bonus Task



Question 1: Why did we have to use console.log( ) and not old friend document.writeln( )?

We had to use console.log( ) so it would appear in our console. Using document.writeln( ) completely wrote over the whole page, which is what we don’t want. Referring back to Lab 7, we did the window prompt and name sorting with console.log( ). We do all this in console.log( ) because we want it to happen on the page internally, behind the scenes, or basically somewhere where the user won’t see it happen. document.writeln( ) makes whatever we’re coding appear on the page.
