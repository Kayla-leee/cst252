LAB SUMMARY

This purpose of this lab was to create a local file structure to organize our files better, and then upload them to the server. After creating the file structure, which was already made for us, we were to create index.html files in our CST 252 and Lab 3 folders. I stylized my pages using CSS, and then uploaded the files to the server using GitHub.
