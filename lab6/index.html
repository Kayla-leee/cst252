LAB 6: ARRAYS AND OBJECTS

LAB SUMMARY

This lab was focused on experimenting with Javascript arrays and objects. According to the lecture, arrays are used to store data in an ordered list, and objects are used to structure that data. Like the previous lab, we used the Console in the browser Inspect to test how the codes would come out as. After that, we moved to Sublime to enter it in the Javascript file.

CHALLENGES

We had no major challenges with this lab, except for forgetting what we had to do. We had to rejog our memories with the lecture slides, as well as using the screenshots on the assignment page to remember how arrays and objects worked. After entering everything into the Javascript file in Sublime, a few things didn't appear. We went into Inspect to figure out what went wrong, and turns out it was an extra or missing semicolon. We're starting to get used to using Inspect to look into issues like these.
