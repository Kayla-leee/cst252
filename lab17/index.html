LAB 17: ADVANCED JQUERY

LAB SUMMARY

This lab was focused on experimenting with advanced jQuery, again by creating and playing around with buttons. We were to create the buttons within HTML this time, and then add a click function to them with jQuery. These buttons are connected to each of the sections, and by clicking the buttons, the corresponding section will change its style. We used removeClass and toggleClass to implement these changes, while going into our CSS files to create the new styles.

CHALLENGES

Our main challenge for this lab was matching everything between all three files, since we got a bit confused between referencing HTML classes and CSS classes in jQuery. For me, since I already had a site CSS set for all labs, I had to remove that class style in order to toggle the new class style. After that, however, repressing the button to untoggle the new class would not retoggle my initial CSS. A simple solution was to retoggle the original class under the same function without needing to remove the new class.
