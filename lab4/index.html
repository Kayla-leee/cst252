How to Shuffle a Deck of Cards (Riffle Shuffle)

  1. Split the deck into two equal piles.
  2. Take one pile into each hand.
  3. Hold the base of the piles with two fingers, and hold the tops with your thumbs.
  4. Bend the piles with your fingers and thumbs, and then release your thumbs, letting the cards move and pile on top of each other.
  5. Combine piles and start again, if needed.

Creating a Rock-Paper-Scissors Game

  1. Create choices: rock, paper, and scissors.
  2. Create an opponent with randomized choices.
  3. Allow user to make a choice.
  4. Allow computer to make a choice.
  5. Create a scenario where the user wins.
  6. Test.

LAB SUMMARY

For this lab, we learned how to pseudocode using a Javascript file, and turning them into comments. We haven't learned how to do anything with Javascript yet, but pseudocoding seems like a start. As for the problem-solving portion, we were to come up with two sets of basic, or "high-level," instructions: how to shuffle a deck of cards, and creating a rock, paper, scissors game on your computer.
