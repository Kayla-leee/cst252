LAB SUMMARY

For this lab, we experimented with loops in Javascript. We created a for loop that printed out the numbers 1-200, where certain numbers would print out a word instead of the number. After making sure it worked in the browser console, we were to print our output onto our page. We came across issues with that, even after reviewing the lecture, using Wes' suggested code from Slack, and asking fellow classmates. I eventually looked back at my previous labs and figured out how to make the output print on the page in a way that made sense to me. It's obviously not the most professional or proper way to make it work, but it worked. For the bonus, styling the output into neat columns was not difficult, as we used CSS attributes from W3Schools to do so.
