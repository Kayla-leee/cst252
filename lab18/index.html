Lab Summary

The focus of this lab was to experiment with Bootstrap. Instead of the usual "div id='content' div class='blahblah'," we were to use containers, rows, and col, which is what makes up the Bootstrap layout.

Challenges

There were a few challenges with this lab. The Bootstrap overwrote my font family and font sizes. I did some research with how to solve this issue, and all I needed to do was be more specific in my CSS. So if I wanted the whole page to be one font, using "body {}" wouldn't work, and I'd have to list out all of my divs, headers, paragraphs, links, navs, titles, etc. and then put the font family under that CSS class. Ashlyn also had issues of her own, which was making the col go side-by-side. She had three col in a row, which naming them "div class='col'" should work, but it didn't for some reason. Her code looked exactly like mine, yet it worked differently. We are still trying to figure that out, so when we solve it for Ashlyn's website, we will resubmit the doc.

Results

The results can be seen with how the page laid out! The summary, challenges, and results are styled with "row," "container," and "col" classes. Open the inspector to look at the HTML and see what we accomplished.
