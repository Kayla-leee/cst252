LAB 11: CONDITIONALS

LAB SUMMARY

This lab was focused on creating a function that depends on conditionals. Like the previous lab, we created an input form field with a button that has a function connected to it. Instead of a Harry Potter sorting hat function, Ashlyn and I decided to put our own twist to it. Ashlyn's function has a conditional that returns the Warrior Cat Clans (4), and mine returns the Eeveelutions (8) of Eevee, the Pokemon. We used Wes' JS code for reference from the screenshots on the lab assignment page.
